# Swift

Initialize the project.

```shell
$ cd years/${YEAR}
$ mkdir swift
$ cd swift
$ swift package init --type tool --name AoC
Creating tool package: AoC
Creating Package.swift
Creating .gitignore
Creating Sources/
Creating Sources/AoC.swift
```

Run the skeleton code.

```shell
$ swift run AoC
...
Build complete! (6.85s)
Hello, world!
```
